## 📍 개요 | 소개

**Wukong Virtual Location**은 개발자와 사용자를 위해 정밀한 지리적 위치 시뮬레이션을 제공하도록 설계된 전문 Android GPS 모의 위치 도구입니다. 루트 권한이 필요하지 않으며 모든 Android 버전을 지원하여 가장 신뢰할 수 있는 가짜 위치 솔루션입니다.

**키워드**: mock gps, fake location, gps spoof, 위치 시뮬레이터, 안드로이드 개발자 도구, LBS 테스팅, GPS 에뮬레이터, 가상 위치, 위치 변조

## ⭐ 주요 기능 | 핵심 특징

### 🎯 정밀한 위치 시뮬레이션
- **루트 권한 불필요**: 모든 Android 기기에서 루트 액세스 없이 작동
- **정확한 좌표**: GPS 및 네트워크 위치 스푸핑 모두 지원
- **실시간 업데이트**: 부드러운 전환과 함께 즉각적인 위치 변경
- **경로 시뮬레이션**: 조정 가능한 속도로 사용자 정의 이동 경로

### 🛠️ 개발자 도구
- **LBS 테스팅 스위트**: 완전한 위치 기반 서비스 테스팅 프레임워크
- **API 호환성**: 모든 주요 위치 API 및 프레임워크와 호환
- **디버그 모드**: 상세한 위치 로그 및 오류 보고
- **배치 테스팅**: 여러 테스트 위치 간의 빠른 전환

### 🔧 고급 옵션
- **조이스틱 컨트롤**: 게임패드 스타일의 정밀한 움직임 제어
- **즐겨찾기 위치**: 자주 사용하는 위치에 원클릭 액세스
- **히스토리 추적**: 완전한 사용 히스토리 및 위치 기록
- **가져오기/내보내기**: GPX, KML 및 기타 표준 형식 지원

### 1. 설치 과정
```bash

APK 다운로드 → 알 수 없는 소스 허용 → 설치
```

1. "설정" → "개발자 옵션" 열기
2. "모의 위치 앱" 활성화
3. "Wukong Virtual Location" 선택
4. 위치 액세스 권한 부여

### 3. 기본 사용법
```
1. 애플리케이션 실행
2. 지도에서 대상 위치 탭
3. "모킹 시작" 클릭

```

## ❓ FAQ | 자주 묻는 질문

### Q: 루트 권한이 필요한가요?
A: 아니요, 이 앱은 시스템 모의 위치 API를 사용하여 비루팅 기기에서 완벽하게 작동합니다.

### Q: 어떤 Android 버전이 지원되나요?
A: Android 4.4부터 최신 Android 14까지 모든 버전이 완전히 지원됩니다.

### Q: 다른 앱에서 탐지될 수 있나요?
A: 고급 후킹 기술을 사용하여 탐지를 극도로 어렵게 만들지만, 책임감 있게 사용하세요.

### Q: 다른 애플리케이션에 영향을 미치나요?
A: 위치 관련 기능에만 영향을 미치며, 다른 모든 앱 기능은 영향받지 않습니다.
