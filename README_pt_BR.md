## 📍 Visão Geral

**Wukong Virtual Location** é uma ferramenta profissional de mock GPS para Android desenvolvida para desenvolvedores e usuários fornecerem simulação precisa de localização geográfica. Não requer root, suporta todas as versões do Android, tornando-se a solução de localização falsa mais confiável disponível.

**Palavras-chave**: mock gps, fake location, gps spoof, simulador de localização, ferramentas de desenvolvedor android, teste LBS, emulador GPS, localização virtual, falsificador de posição

## 🚀 Guia de Início Rápido

### 1. Processo de Instalação
```bash
# Método 1: Instalação Direta de APK
Baixar APK → Habilitar Fontes Desconhecidas → Instalar

# Método 2: Instalação para Desenvolvedores
adb install MockLocation.apk
```

### 2. Configuração de Permissões
1. Abrir "Configurações" → "Opções do Desenvolvedor"
2. Habilitar "App de Localização Mock"
3. Selecionar "Wukong Virtual Location"
4. Conceder permissões de acesso à localização

### 3. Uso Básico
```
1. Iniciar o aplicativo
2. Tocar na localização desejada no mapa
3. Clicar em "Iniciar Mock"
4. Verificar se a localização está ativa
```

## ❓ FAQ

### P: É necessário acesso root?
R: Não, este app funciona perfeitamente em dispositivos sem root usando a API de localização mock do sistema.

### P: Quais versões do Android são suportadas?
R: Todas as versões do Android de 4.4 até o mais recente Android 14 são totalmente suportadas.

### P: Pode ser detectado por outros apps?
R: Usa tecnologia avançada de hook tornando a detecção extremamente difícil, mas use responsavelmente.

### P: Afeta outras aplicações?
R: Afeta apenas funções relacionadas à localização, todas as outras funcionalidades do app permanecem inalteradas.
